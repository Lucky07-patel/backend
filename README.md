# Lead Capture Frontend

This is the frontend for the Lead Capture project built using React.js, Vite, and Tailwind CSS.

## Features

* Responsive lead capture form
* Lead submission functionality
* Fields:

  * Full Name
  * Email
  * Business Name
  * Message
* Client-side validation
* Error messages for invalid inputs
* Admin dashboard to view submitted leads
* AI Lead Qualification Status:

  * Hot
  * Warm
  * Cold
  * Pending
* Responsive UI for desktop and mobile devices

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* JavaScript (ES6+)
* Fetch API

## GitHub Repository

Clone the repository:

```bash
git clone https://github.com/your-username/lead-capture-frontend.git
```

Navigate to the project folder:

```bash
cd lead-capture-frontend
```

---

## Setup Instructions

### 1. Install Dependencies

```bash
npm install
```

### 2. Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=http://localhost:5000
```

For production:

```env
VITE_API_URL=https://your-backend-url.com
```

---

### 3. Run Development Server

```bash
npm run dev
```

The application will run at:

```bash
http://localhost:5173
```

---

### 4. Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## API Integration

The frontend communicates with the backend API for:

### Submit Lead

```http
POST /leads
```

### Fetch All Leads

```http
GET /leads
```

### Dashboard Statistics

```http
GET /leads/dashboard
```

## Notes

* Update the `VITE_API_URL` environment variable according to your backend URL.
* If the backend is running locally, ensure PostgreSQL and the backend server are active.
* AI lead scoring is generated by the backend using the Groq AI API.
* The project follows a component-based architecture for maintainability and scalability.
